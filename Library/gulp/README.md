# Gulp
 - Gulp는 JavaScript 기반의 Task Runner(작업 자동화 도구) 이다.
 - 반복적으로 수행하는 개발 작업을 자동화하여 개발 생산성을 높여준다.
 - 파일을 Stream 방식으로 처리하기 때문에 Grunt보다 빠른 성능을 제공한다.
 - Webpack, Vite 등의 등장으로 사용 빈도는 줄었지만, 레거시 프로젝트에서는 여전히 많이 사용된다.

 
## 설치
 - npm i gulp
 - npm i gulp-uglify (코드 압축을 위한 설치)
 - npm i gulp-insert (압축된 코드에 주석 작성을 위한 설치)
 - pnpm add gulp
 - pnpm add gulp-uglify (코드 압축을 위한 설치)
 - pnpm add gulp-insert (압축된 코드에 주석 작성을 위한 설치)
 - npm i -D @types/gulp (typescript 환경일시)
 - pnpm add -D @types/gulp (typescript 환경일시)


## 명령어 
 - gulp dev
