# Hono 
 - Hono는 Web Standards 기반의 초경량(ultrafast) 웹 프레임워크이다.
 - JavaScript와 TypeScript를 모두 지원하며, Express보다 가볍고 빠른 성능을 목표로 만들어졌다.
 - Node.js뿐만 아니라 Cloudflare Workers, Bun, Deno, Vercel, AWS Lambda 등 다양한 런타임에서 동일한 코드로 실행할 수 있다.
 - Express처럼 라우팅을 제공하지만, Fetch API(Request, Response) 를 기반으로 동작하는 것이 가장 큰 특징이다.
 - TypeScript(Node.js) 환경에 최적화된 경량 웹 프레임워크이다.
