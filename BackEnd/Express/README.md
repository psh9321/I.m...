# Express
 - Express는 Node.js에서 가장 널리 사용되는 웹 애플리케이션 프레임워크이다. 
 - HTTP 서버를 쉽게 구축할 수 있도록 라우팅(Routing), 미들웨어(Middleware), 요청/응답 처리 기능을 제공한다.
 - 내부적으로 Node.js의 http 모듈을 기반으로 동작하며, 복잡한 서버 코드를 간결하게 작성할 수 있다.
 - REST API 서버, SSR 서버, 파일 업로드 서버 등 다양한 웹 애플리케이션을 구축하는 데 사용된다.
 - JavaScript(Node.js) 환경에 최적화된 경량 웹 프레임워크이다.
 - TypeScript도 지원하지만 별도의 타입 정의(@types/express)를 사용한다.


## 설치
 - npm i express
 - pnpm add express
