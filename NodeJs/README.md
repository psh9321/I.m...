# Node.js
 - Node.js는 Chrome V8 JavaScript Engine 기반의 JavaScript 런타임(Runtime)이다.
 - JavaScript를 브라우저뿐만 아니라 서버 환경에서도 실행할 수 있도록 해준다.
 - 이벤트 루프(Event Loop) 기반의 비동기 I/O를 사용하여 많은 요청을 효율적으로 처리할  수 있다.
 - npm(Node Package Manager)을 통해 다양한 패키지를 사용할 수 있다.


## 특징
 - JavaScript를 서버 환경에서 실행할 수 있다.
 - 단일 스레드(Single Thread) 기반으로 동작한다.
 - 이벤트 루프(Event Loop)를 통해 비동기 작업을 처리한다.
 - npm을 통해 라이브러리와 프레임워크를 쉽게 관리할 수 있다.


## 주요 활용
 - REST API 서버
 - SSR(Server Side Rendering)
 - BFF(Backend For Frontend)
 - CLI(Command Line Interface)
 - Build Tool(Webpack, Vite 등)
