# NginX 기본문법


### Nginx 설정 파일 수정
 - cd /etc/nginx/conf.d 로 접근
 - sudo vi default.default.conf

```
server_name {{이 서버블록이 처리할 도메인}} 

# / 이하의 모든 요청을 처리
location / { 
    proxy_pass http://127.0.0.1:{PORT 번호}; # server_name 으로 온 요청을 proxy_pass 로 대신 보내줌 
    }
```


### Nginx 파일 테스트 명령어
 - sudo nginx -t : nginx 설정파일에 문제가있는지 테스트하는 명령어
 - 문제가 없을시 ~~ test is successful


### 설정 파일을 수정 후 반영하기위한 명령어 
 - sudo systemctl reload nginx : systemd에게 "Nginx를 reload해"라고 요청
 - sudo nginx -s reload : Nginx 마스터 프로세스에게 직접 reload 신호(HUP)를 보냄
