# Linux
 - Linux는 유닉스(Unix) 계열의 오픈소스 운영체제(OS) 이다.
 - 서버, 클라우드, 임베디드, 모바일(Android) 등 다양한 환경에서 사용된다.
 - 멀티태스킹, 멀티유저를 지원하며 안정성과 보안성이 뛰어나 서버 운영체제로 가장 많이 사용된다.
 - 대부분의 웹 서버(Nginx, Apache), Node.js, Docker, Kubernetes 등이 Linux 환경에서 실행된다


## 주요 디렉터리
| 디렉터리    | 설명                   |
| ------- | -------------------- |
| `/`     | 루트(Root) 디렉터리        |
| `/home` | 일반 사용자 홈 디렉터리        |
| `/root` | root 사용자 홈 디렉터리      |
| `/etc`  | 시스템 설정 파일            |
| `/var`  | 로그, 캐시 등 자주 변경되는 데이터 |
| `/usr`  | 프로그램 및 라이브러리         |
| `/bin`  | 기본 실행 명령어            |
| `/sbin` | 시스템 관리 명령어           |
| `/tmp`  | 임시 파일                |
| `/opt`  | 추가 설치 프로그램           |


## 명령어
 - pwd : 현재 위치 확인
 - ls : 목록 조회
 - cd : 디렉터리 이동
 - mkdir : 디렉터리 생성
 - touch : 파일 생성
 - cp : 복사
 - mv : 이동 및 이름 변경
 - rm : 삭제
 - find : 파일 검색


## 내가 자주 사용한 명령어
 - cat {{파일명}} : 파일 내용 출력
 - vi {{파일명}} : 파일 수정 및 생성
 - rm -rf. {파일/디렉터리 명} : 삭제 (-rf 가 붙을시 디렉터리 확인 없이 모든 파일과 하위 디렉터리까지 강제로 전부 삭제)
 - cd {경로} : 디렉터리 이동
