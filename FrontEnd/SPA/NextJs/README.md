# NextJs
 - React 기반의 풀스택 웹 프레임워크
 - 서버 사이드 렌더링(SSR), 정적 사이트 생성(SSG), 증분 정적 생성(ISR), 클라이언트 사이드 렌더링(CSR)을 지원한다.
 - 파일 시스템 기반 라우팅을 제공하며, App Router를 통해 Server Component와 Client Component를 사용할 수 있다.
 - API Route(Route Handler), Middleware 등 서버 기능을 함께 제공하여 BFF(Backend For Frontend) 구조를 구현할 수 있다.
 - 이미지 최적화, 폰트 최적화, 코드 분할(Code Splitting) 등 다양한 성능 최적화 기능을 기본 제공한다.
