# SSR (Server Side Rendering)
 - 사용자의 요청이 들어올 때마다 서버에서 HTML을 생성하여 브라우저에 반환하는 렌더링 방식이다.
 - 브라우저는 완성된 HTML을 먼저 화면에 표시한 뒤, JavaScript를 내려받아 Hydration 과정을 거쳐 상호작용이 가능한 페이지로 만든다.
 - 초기 화면 표시가 빠르고 검색 엔진이 HTML을 바로 수집할 수 있어 SEO에 유리하다.


## 동작 과정
 1. 브라우저가 서버에 페이지를 요청한다.
 2. 서버가 필요한 데이터를 조회한다.
 3. React 컴포넌트를 HTML로 렌더링한다.
 4. 완성된 HTML을 브라우저에 응답한다.
 5. 브라우저가 HTML을 먼저 화면에 표시한다.
 6. JavaScript를 다운로드한 후 Hydration을 수행하여 이벤트를 연결한다.
