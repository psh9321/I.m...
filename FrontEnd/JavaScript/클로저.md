# 클로저 (Closure)
 - 함수가 선언될 당시의 렉시컬 환경(Lexical Environment)을 기억하여, 
   외부 함수가 종료된 이후에도 그 변수에 접근할 수 있는 기능이다.

## 함수를 반환만 했을뿐 실행하지않은 상태
```javascript
    function outer() {
        const name = "su hyun";

        function inner() {
            console.log(name);
        }

        return inner;
    }

    const my = outer(); 

    console.log(my);
    /**
     * ƒ inner() {
     *    console.log(name);
     * }
    */

    /**
     * 반환된 함수(inner)를 console.log로 확인만 했을 뿐,
     * 실행(my())하지 않았기 때문에 "su hyun"은 출력되지 않는다.
    */
```


## 함수를 반환받고 실행한 경우
```javascript
    function outer() {
        const name = "su hyun";

        function inner() {
            console.log(name);
        }

        return inner;
    }

    const my = outer(); /** inner 함수를 반환 */

    my(); /** su hyun */

```

### 호출 순서
```javascript
    outer()
    ↓
    name = "su hyun"
    ↓
    return inner
    ↓
    my = inner
    ↓
    outer 종료
    (렉시컬 환경은 inner가 참조하므로 유지)
    ↓
    my()
    ↓
    console.log(name)
    ↓
    /** su hyun */
```
