# 객체 (Object)
 - 여러 개의 데이터를 Key : Value 형태로 저장하는 자료형이다.
 - 객체 내부의 Key와 Value 한 쌍을 Property(프로퍼티)라고 한다.
 - JavaScript는 객체 기반(Object-Based) 언어이며, 
   Array, Function, Date 등도 모두 Object이다.
 

## 생성 방법 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    const user2 = new Object();

    user2.name = "su hyun2";
    user2.firstName = "Park2";
```

## 프로퍼티 접근 방법 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    console.log(user.name) /** su hyun */
    console.log(user["firstName"]) /** "Park" */
```

## 프로퍼티 추가 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    user.job = "developer";
    user["annual"] = 4;

    console.log(user) /** { name : "su hyun", firstName : "Park", job : "developer", annual : 4 } */
```

## 프로퍼티 제거 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    delete user.firstName;

    console.log(user) /** { name : "su hyun" } */

```

## 프로퍼티 확인 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    console.log("name" in user) /** true */
    console.log("job" in user) /** false */
```

## 객체 비교 
```javascript
    const user1 = {
        name: "su hyun"
    };

    const user2 = {
        name: "su hyun"
    };

    console.log(user1 === user2); /** false */
    console.log(user1.name === user2.name); /** true */
```

```javascript
    const user1 = {
        name: "su hyun"
    };

    const user2 = user1;

    console.log(user1 === user2); /** true */
```

## 구조분해할당
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    const { firstName, name } = user;

    console.log(firstName) /** Park */
    console.log(name) /** su hyun */


```


## 자주 사용하는 객체 메서드 

### keys : 프로퍼티 키만 반환
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    console.log(Object.keys(user)); /** ["name", "firstName"] */
```


### values : 프로퍼티 value 만 반환 
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    console.log(Object.values(user)); /** ["su hyun", "Park"] */
```

### entries : 프로퍼티 key 와 value 를 배열에 담아 반환
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }

    console.log(Object.entries(user)); /** [["name","su hyun"], ["firstName","Park"]] */
```

### 객체 병합 : 기존값을 복사해 병합
 - assign : assign : 객체를 병합하거나 복사한다. 
            첫 번째 인자로 전달한 객체를 변경한다.
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }    

    Object.assign(user, {job : "developer"});

    console.log(user) /** { name : "su hyun", firstName : "Park", job : "developer" } */
```

- spread : 원본 객체를 변경하지 않고 새로운 객체를 생성하여 병합한다.
```javascript
    const user = {
        name : "su hyun",
        firstName : "Park"
    }    

    const newUser = {
        ...user,
        job : "developer"
    }
    console.log(newUser) /** { name : "su hyun", firstName : "Park", job : "developer" } */
```
