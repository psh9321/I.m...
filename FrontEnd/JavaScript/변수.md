# 변수 (Variable)
 - 데이터를 저장하기 위한 메모리 공간이다.
 - 변수명을 통해 저장된 데이터에 접근하고 사용할 수 있다.


## let 
 - 재선언 x
 - 재할당 o

## const 
 - 재선언 x
 - 재할당 x
 - 재할당은 막지만 내부의 값은 변경가능
 ```javascript
    const arr = [1,2,3];

    arr.push(4);

    console.log(arr); /** [1,2,3,4] */

    const obj = {
        name : "su hyun"
    }

    obj["firstName"] = "Park";

    console.log(obj) /** { name : "su hyun", firstName : "Park" } */
 ```

## var 
 - 재선언 o
 - 재할당 o

### var 의 문제점 
 - let, const 와달리 스코프 범위 불명확해 블록을 벗어나도 접근이 가능하다. 
 ```javascript
    if(true) {
        var a = 10;
    }

    console.log(a) /** 10 */
 ```

 ```javascript
    if(true) {
        let a = 10;
    }
    
    console.log(a) /** ReferenceError */
 ```
- 호이스팅
```javascript
    console.log(a); /** 생성단계에서 a 가 undefined 로 초기화 되고 undefined 출력 */

    var a = 10;

```

```javascript
    console.log(a); /** 생성 단계에서 선언 되었지만 ReferenceError */

    const a = 10;
```

| 구분    | var                | let        | const   |
| -----  | ----------------   | --------- | -------  |
| 재선언   | ⭕                 | ❌        | ❌        |
| 재할당   | ⭕                 | ⭕        | ❌        |
| 스코프   | 함수 스코프           | 블록 스코프 | 블록 스코프  |
| 호이스팅  | `undefined`로 초기화 | TDZ      | TDZ         |
| 사용 권장 | ❌                 | ⭕        | ⭐ 기본 사용  |
