# 배열 (Array)
 - 여러 개의 값을 하나의 변수에 저장하기 위한 특수한 객체이다.
 - 배열은 객체(Object)의 한 종류이므로 typeof의 결과는 object이다. 
   Array.isArray 를 사용해 배열 확인이 가능하다.
```javascript
    const arr = [1,2,3];

    console.log(typeof arr) /** object */

    console.log(Array.isArray(arr)) /** true */
```


## 생성 방법 
```javascript
    const arr = [1,2,3];

    const arr2 = new Array(1,2,3);
```

## 요소 접근 
 - 0 부터 시작
```javascript
    const arr = [1,2,3];

    console.log(arr[0]) /** 1 */
    console.log(arr[1]) /** 2 */
    console.log(arr[2]) /** 3 */ 
```

## 길이 
```javascript
    const arr = [1,2,3];

    console.log(arr.length) /** 3 */
```

## 요소 추가 

### push : 뒤에 추가
 ```javascript
    const arr = [1,2,3];

    arr.push(4); /** 뒤에 추가 */

    console.log(arr); /** [1,2,3,4] */

 ```


### unshift : 앞에 추가
 ```javascript
    const arr = [1,2,3];

    arr.unshift(4); /** 앞에 추가 */

    console.log(arr); /** [4,1,2,3] */
 ```

## 요소 제거 

### pop : 뒤에서 제거
```javascript
    const arr = [1,2,3];

    arr.pop();

    console.log(arr) /** [1,2] */
```


### shift : 앞에서 제거
```javascript
    const arr = [1,2,3];

    arr.shift();

    console.log(arr) /** [2,3] */
```


## 구조분해할당
```javascript
    const arr = [{a : 1},{b : 2},{c : 3}];

    const [a, b, c] = arr;

    console.log(a) /** {a : 1} */
    console.log(b) /** {b : 2} */
    console.log(c) /** {c : 3} */

```


## 자주 사용하는 배열 메서드

### map : 새로운 배열을 생성
 ```javascript
    const arr = [1,2,3];

    const result = arr.map(el=>el*2);

    console.log(result) /** [2,4,6] */;
 ```


### filter : 조건에 맞는 요소만 반환 
 ```javascript
    const arr = [1,2,3,4];

    const result = arr.filter(el=>el%2===0) /** [2,4] */;
 ```

### find : 첫번째 요소 하나 반환
```javascript
    const arr = [1,2,3,4];

    const result = arr.find(el=>el>2);

    console.log(result) /** 3 */
```

### some : 배열안의 요소가 조건과 하나라도 맞으면 true 를 반환
```javascript
    const arr = [1,2,3,4];

    const result = arr.some(el=>el<2);

    console.log(result) /** true */
```

### every : 배열안의 요소가 조건과 모두 맞아야 true 로 반환
```javascript
    const arr = [1,2,3,4];

    const result = arr.every(el=>el>0);

    console.log(result) /** true */
```

### includes : 조건에 포함하는 요소가 배열안에 있을시 true 를 반환 
```javascript
    const arr = [1,2,3,"string"];

    console.log(arr.includes("string")) /** true */
```

### reduce : 배열안 요소를 하나의 값으로 계산 
```javascript
    const arr = [1,2,3];

    const result = arr.reduce((acc, cur)=>{
        return acc + cur;
    },0);

    console.log(result) /** 6 */
```

### sort : 배열안 요소 정렬
 - sort 는 기본적으로 문자열 기준으로 정렬한다.
 - 숫자 정렬은 비교 함수를 전달해야한다
```javascript
    const arrNum = [5,1,10];

    arrNum.sort((a,b)=>a-b);

    console.log(arrNum) /** [1,5,10] */

    const arrStr = ["b","c","a"];

    arrStr.sort();

    console.log(arrStr) /** ["a","b","c"] */
```

### indexOf, findIndex : 조건에 맞는 첫 번째 요소의 인덱스번호를 반환
 - indexOf : 원시값과 일치하는 첫번째 요소의 인덱스번호를 반환
 - findIndex : 조건에 만족하는 첫번째 요소의 인덱스번호를 반환
```javascript
    const indexOfArr = [1,2,3];

    console.log(indexOfArr.indexOf(2)) /** 1 */

    const findIndexArr = [{sequence : "first"}, {sequence : "second"}, {sequence : "third"}];

    console.log(findIndexArr.findIndex(el => el.sequence === "second")) /** 1 */
```
