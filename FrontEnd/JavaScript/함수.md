# 함수 (Function)
 - 특정 작업을 수행하는 코드의 묶음이다.
 - 필요할 때 여러 번 호출하여 재사용할 수 있다.

## 생성 방법 

### 함수 선언문 (Function Declaration)
```javascript
    function hello() {
        console.log("Hello");
    }
```


### 함수 표현식(Function Expression)
```javascript
    const hello2 = function() {
        console.log("Hello");
    }
```

### 화살표 함수(Arrow Function) 
```javascript
    const add = (a, b) => {
        return a + b;
    };

    const add2 = (a,b) => a+b;

    add(1,2) /** 3 */
    add2(3,4) /** 7 */
```


## 매개 변수와 인수
 - 매개변수(Parameter) : 함수를 선언할 때 사용하는 변수
 - 인수(Argument) : 함수를 호출할 때 전달하는 값
 ```javascript
    function hello(name) { /** Parameter */
        console.log(name);
    }

    hello("su hyun"); /** Argument */
 ```


## return (반환값) : 함수가 실행을 끝내고 돌려주는 값
```javascript
    function add(a, b) {
        return a + b;
    }

    const result = add(10, 20);

    console.log(result); /** 30 */
```


## 기본 매개변수(Default Parameter) 
```javascript
    function hello(name = "Guest") {
        console.log(name);
    }

    hello(); /** Guest */
    hello("su hyun"); /** su hyun */
```


## 나머지 매개변수(Rest Parameter)
```javascript
    function sum(...numbers) {
        console.log(numbers);
    }

    sum("su hyun", "Park", { job : "developer" }); /** ["su hyun", "Park", { job : "developer" }]  */
```


## 익명 함수 : 이름이 없는 함수
```javascript
    const hello = function () {
        console.log("Hello");
    };
```


## 즉시 실행 함수 (IIFE)
 - 선언과 동시에 즉시 실행
```javascript
    (function() {
        console.log("hello")
    })()
```


## 콜백 함수 (Callback)
 - 다른 함수의 인수로 전달되는 함수
```javascript
    function hello(callback) {
        callback();
    }

    hello(() => {
        console.log("callback");
    });
```


## 고차 함수(Higher Order Function)
 - 함수를 인수로 받거나 함수를 반환하는 함수
 - Array 의 map, filter, reduce 도 대표적인 고차함수이다.
 ```javascript
    function hello(callback) {
        callback();
    }
 ```
