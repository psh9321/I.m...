# 이벤트 루프 
 - 단일 스레드인 자바스크립트에서 비동기 작업을 처리할 수 있는 메커니즘 

## 콜 스텍(Call Stack)
 - JavaScript 가 실행 되는 공간 
 - 함수가 호출 되면 스택에 쌓이고, 끝나면 빠져나감 
 - 동기 코드 실행 담당 


## Web Apis (브라우저/Node.js 백그라운드)
 - setTimeout, fetch, DOM 이벤트, Timer, HTTP 요청 등 JS 엔진이 처리하지 않고 
   브라우저(Web APIs) / Node.js(Node APIs)가 비동기 작업을 대신 수행하는 영역


## 콜백 큐 (Callback Queue, Task Queue)
 - 마이크로태스크 큐 (Microtask Queue) : Promise Callbacks (then, catch, finally) 같은 작은 단위의 비동기 작업이 들어가는 큐 
 - 매크로태스크 큐 (Macrotask Queue) : setTimeout, setInterval 등 큰 단위 비동기 작업이 들어가는 큐
 - 매크로 태스크 실행 전 항상 마이크로태스크를 먼저 다 비움


### 실행 순서 
1. 콜스택이 비었는지 확인
2. 마이크로태스크 큐를 모두 비움 
3. Rendering (필요 시)
4. 매크로태스크 큐에서 하나 실행
5. 다시 1번으로 돌아가 반복

```javascript
    console.log(1);

    setTimeout(() => {
        console.log(2);
    }, 0);

    Promise.resolve().then(() => {
        console.log(3);
    });

    console.log(4);

    /**
     * 1
     * 4
     * 3
     * 2
    */
```
