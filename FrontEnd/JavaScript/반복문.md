# 반복문 (Loop)
 - 같은 코드를 여러 번 반복 실행하기 위한 문법이다.
 - 반복 횟수나 순회 대상에 따라 다양한 반복문을 사용할 수 있다.


## for : 반복 횟수가 정해져 있을때 주로 사용한다. 
```javascript
    for (let i = 0; i < 3; i++) {
        console.log(i);
    }

    /**
     * 0
     * 1
     * 2
    */
```

## break : 반복문을 즉시 종료한다.
```javascript
    for (let i = 0; i < 5; i++) {
        if (i === 3) break;

        console.log(i);
    }

    /**
     * 0
     * 1
     * 2
    */
```


## continue : 현재 반복만 건너뛰고 다음 반복을 실행한다.
```javascript
    for (let i = 0; i < 5; i++) {
        if (i === 2) continue;

        console.log(i);
    }

    /**
     * 0
     * 1
     * 3
     * 4
    */
```
 
## while : 조건이 참인 동안 반복한다.
```javascript
    let i = 0;

    while (i < 3) {
        console.log(i);
        i++;
    }

    /**
     * 0
     * 1
     * 2
    */
```

## do while : 조건과 상관없이 최소 한 번은 실행된다.
```javascript
    let i = 5;

    do {
        console.log(i);
    } while (i < 3);

    /** 5 */
```

## for of : 반복 가능한 객체의 값을 순회 한다.
```javascript
    const arr = [1, 2, 3];

    for (const value of arr) {
        console.log(value);
    }
 
    /**
     * 1
     * 2
     * 3
    */

    for (const str of "Park") {
        console.log(str);
    }

    /**
     * P
     * a
     * r
     * k
    */
```

## for in : 객체 프로퍼티(Key)를 순회한다.
```javascript
    const user = {
        name: "su hyun",
        firstName: "Park"
    };

    for (const key in user) {
        console.log(key, user[key]);
    }

    /**
     * name su hyun
     * firstName "Park"
    */
```

## forEach : 배열의 각 요소를 순회한다.
 - break, continue 를 사용할수없다.
```javascript
    const arr = [1, 2, 3];

    arr.forEach((value, index) => {
        console.log(value, index);
    });

    /**
     * 1 0
     * 2 1
     * 3 2
    */
```

| 반복문          | 주 사용 대상                            |
| ------------ | ---------------------------------- |
| `for`        | 반복 횟수가 정해진 경우                      |
| `while`      | 조건이 만족하는 동안 반복                     |
| `do...while` | 최소 한 번은 실행해야 하는 경우                 |
| `for...of`   | Array, String, Map, Set 등 Iterable |
| `for...in`   | Object의 Key 순회                     |
| `forEach()`  | Array 순회                           |
