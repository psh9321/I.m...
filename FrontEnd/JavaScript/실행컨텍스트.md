## 실행 컨텍스트 (Execution Context)
 - JavaScript 코드가 실행되기 위해 필요한 정보 (변수, 함수, 스코프, this 등)를 저장하고 관리하는 실행 환경
 - 프로그램이 시작되면 가장 먼저 생성
 - Execution Context는 Call Stack에 저장된다

### 생성 순서 

1. 생성 단계(Creation Phase) : 코드를 실행 하기전에 필요한 정보를 먼저 준비
 - 변수 선언 등록
 - 함수 선언 등록
 - 렉시컬 환경(Lexical Environment) 생성
 - 현재 실행 컨텍스트의 this 바인딩 결정


2. 실행 단계 (Execution Phase)
```javascript
    const a = 10; /** a 에 10 저장 */

    console.log(a); /** 10 출력 */
```


### Function Execution Context
 - 함수를 호출할 때마다 새로 생성
```javascript
    function hello() {
        console.log("Hello");
    }

    hello();

    /**
    * 실행 순서
    * 1. Global Context 생성
    * 2. hallo() 호출
    * 3. hello Context 생성
    * 4. hello 종료
    * 5. hello Context 제거
    */
```


### Callstack과의 관계
- 함수가 종료되면 해당 실행 컨텍스트도 CallStack 에서 제거
```javascript
    function a() {
        b();
    }

    function b() {
        c();
    }

    function c() {
        console.log("끝");
    }

    a();

    /**
    * 실행 순서
    * Global // Global Context 생성
    * ↓ 
    * a() // a호출
    * Global
    * ↓
    * b() // b호출
    * a() 
    * Global
    * ↓
    * c() // c호출
    * b()
    * a() 
    * Global
    * ↓
    * // c종료
    * b() 
    * a() 
    * Global
    * ↓
    * //b종료
    * a() 
    * Global
    * ↓
    * //a종료 
    * Global
    */
```


### 내부구조 (Scope Chain)
Execution Context

├── Lexical Environment
│     ├── 변수
│     ├── 함수
│     └── 스코프 정보
│
├── this
│
└── Outer Environment
      (상위 스코프 참조)

- Lexical Environment : 현재 스코프의 변수와 함수 선언을 저장한다.
```javascript
    function hello() {
        const name = "su hyun";
    }

    /** name은 hello의 Lexical Environment에 저장된다. */
```

- this : 현재 실행 중인 this 값을 저장한다
```javascript
    const obj = {
        name: "su hyun",
        hello() {
            console.log(this.name); /** su hyun */
        },
    };

    obj.hello() /** hello 가 실행될때 this 는 obj */
```

- Outer Environment : 상위 스코프를 참조한다. 
```javascript
    const name = "su hyun";

    function hello() {
        console.log(name);
    }
    /**
    * hello 내부에 name 이 없으므로, 렉시컬 환경에서 찾지 못하면 Outer Environment 를 따라 전역 스코프를 탐색한다.
    */
```
